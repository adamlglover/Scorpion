Welcome to Scorpion.

Scorpion is a dynamic bare metal programming language supporting various programming paradigms. It provides a set of features that allow for easy text and data processing as well as programming the physical hardware on a specific device. It is very simple and easy to use. Scorpion is intended to be used for embedded systems and consumer applications that runs on micro-electronics. Check out the repo:

$ cd your_root_dir/Github
$ git clone https://github.com/AndroDevcd/Scorpion/
$ cd Scorpion/
$ sudo cp "bin/linux_debian/raspberryPi/0.0.19_19 beta" /bin
$ sudo scorpion -help

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Why Scorpion?

Scorpion was designed to simplify low level assembly language programming. It provides a lot of the features you would get in a low level assembly language while still supporting some essential high level features as well. Scorpion has a wide range of benefits such as:

  • A very Simple Syntax.
  • Easy to learn. Scorpion was designed to be easy to use and is therefore easy to write, compile, debug, and learn than many other traditional assembly languages.
  • Fast execution time of programs(1.5Mhz processor speed).
  • Support for low level programming of physical computer hardware.
  • Somewhat object-oriented. This allows you to create modular programs and reusable code.
  • Scorpion is platform-independent.
  • Open source under the Apache License, Version 2.0.
